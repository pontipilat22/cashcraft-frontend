================================================================================
                        CASHCRAFT3 - РЕАЛИЗОВАННЫЕ ФУНКЦИИ
================================================================================

УПРАВЛЕНИЕ СЧЕТАМИ
- Создание счетов: наличные, дебетовые/кредитные карты, накопления, кредиты
- Редактирование и удаление счетов
- Установка счета по умолчанию
- Включение/исключение из общего баланса
- Отображение последних 4 цифр карты
- Расчет кредитов (аннуитетный/дифференцированный платеж)

ТРАНЗАКЦИИ
- Добавление доходов и расходов
- Переводы между счетами
- Категоризация операций
- Редактирование и удаление транзакций
- История операций с фильтрацией
- Автоматическое обновление балансов

КАТЕГОРИИ
- Предустановленные категории (11 штук)
- Создание пользовательских категорий
- Редактирование категорий
- Удаление с переносом транзакций
- Иконки и цвета для категорий

УПРАВЛЕНИЕ ДОЛГАМИ
- Учет долгов "Я должен" и "Мне должны"
- Установка даты возврата
- Частичное погашение
- Полное погашение с созданием транзакции
- Включение/исключение из общего баланса
- Отображение просроченных долгов

АУТЕНТИФИКАЦИЯ
- Регистрация и вход по email/паролю
- Apple Sign In (iOS)
- Гостевой режим
- Автоматический вход
- Профиль пользователя
- Выход из системы

РАЗДЕЛЕНИЕ ДАННЫХ
- Отдельная SQLite база для каждого пользователя
- Изолированное хранилище AsyncStorage
- Автоматическая загрузка данных при входе
- Очистка данных при выходе

PREMIUM ПОДПИСКА
- Месячный план: $2/месяц
- Годовой план: $15/год
- Неограниченные счета (бесплатно до 3)
- Экспорт данных
- Расширенная аналитика
- Синхронизация устройств
- Приоритетная поддержка
- Кастомные темы

БАЗА ДАННЫХ
- SQLite с 4 таблицами
- Автоматическая миграция схемы
- Транзакционная целостность
- Подготовка к синхронизации (syncedAt)

UI/UX
- Bottom Tab навигация
- Модальные окна для форм
- Pull-to-refresh
- Swipe-to-delete
- Анимированные переходы
- Toast уведомления
- Адаптивный дизайн
- Splash screen

ДОПОЛНИТЕЛЬНО
- Поддержка 4 валют (RUB, USD, EUR, GBP)
- Светлая тема (темная подготовлена)
- Главный экран с быстрыми действиями
- Экран настроек
- Сброс всех данных
- Валидация входных данных
- Защита от SQL-инъекций

ТЕХНОЛОГИИ
- React Native + Expo
- TypeScript
- SQLite (expo-sqlite)
- React Context API
- React Navigation
- AsyncStorage
- React Native Elements
- Ionicons
