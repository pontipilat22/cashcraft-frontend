# Интерактивный График Баланса

## Функциональность

✅ **Интерактивное отслеживание** - сумма меняется при движении курсора по графику
✅ **Случайные данные** - генерация реалистичных данных для разных периодов
✅ **Переключение периодов** - 6 периодов: 0M, 1W, 1M, 3M, 1Y, 4Y
✅ **Анимация** - плавная анимация при загрузке и смене периода
✅ **Визуальные индикаторы** - точки и линии при наведении

## Установка

1. Скопируйте файлы в папку вашего проекта:
```bash
C:\project\charttest\app\
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект:
```bash
npm run dev
```

## Использование в вашем проекте

```tsx
import Chart from './chart';

function App() {
  return <Chart />;
}
```

## Особенности

- **Адаптивный дизайн** - работает на всех устройствах
- **TypeScript** - полная типизация
- **Tailwind CSS** - для стилизации
- **React Hooks** - современный подход

## Настройка

Вы можете изменить:
- Начальную сумму (сейчас $11,950)
- Цвета графика
- Количество точек данных
- Волатильность для разных периодов
